					               <div class="timeline-label">
					                  <h2><a href="#">A rule-based Decision Support System in Intelligent Hazmat Transportation Systems</a></h2>
					                  <p>To improve the performance of accident severity prediction, we investigate rule extraction methods from decision trees and learning extracted rules with neural network. During this project I work with various techniques of decision trees for generating if-then rules, sparse rules with neural networks, and data mining approaches on accident data. The results is published in IEEE Transaction on ITS 2015, <a href="https://ieeexplore.ieee.org/abstract/document/7097030">here</a></p>.

					               <div class="timeline-label">
					                  <h2><a href="#">A Spatial-Temporal Decomposition Based Deep Neural Network for Time Series Forecasting</a></h2>
					                  <p>In this paper, we formulate spatio-temporal forecasting problem. We design a deep learning model which efficiently consider similarities in spatial data, various periodic patterns in temporal data, and is robust to missing data. The implementations are with Python-Keras. During this project, I work with various techniques for time series forecasting, time series decompositions, and convolutional-recurrent neural networks. The results is published in <a href="https://arxiv.org/pdf/1902.00636.pdf">here</a></p>.

					               <div class="timeline-label">
					               	<h2><a href="#">A convolutional recurrent autoencoder for spatio-temporal missing data imputation.</a></h2>
					                  <p> In this project, we design a convolutional-LSTM neural network for missing data imputation of spatio-temporal data. We illustrate that convolutional-LSTM architectures are stronger models to capture spatial and temporal patterns together. The model efficiently impute missing values from such datasets. During this project, I work with various baseline missing data imputation techniques, convolutional and recurrent layers, and latent feature representation of authoencoders. The results is published in International Conference on AI, 2019 <a href="https://arxiv.org/pdf/1904.12413.pdf">here</a> and 
									   <a href=" https://csce.ucmss.com/cr/books/2019/LFS/CSREA2019/ICA2374.pdf">"A convolutional recurrent autoencoder for spatio-temporal missing data imputation"; Reza Asadi, Amelia Regan; 21st International conference on artificial intelligence;</a></p>.

					                  <p>We developed distributed optimization algorithm for optimal network flow problem, by reducing the number of decision variables using the graph theory. The model has a tremendous improvement compared with classifical optimal network flow methods. The results is published in <a href="https://ieeexplore.ieee.org/abstract/document/7852302">here</a></p> Also, the application of such solution on power grid networks is presented and accept in .  <a href="https://ieeexplore.ieee.org/stamp/stamp.jsp?arnumber=8823582">"Cycle flow formulation of Optimal network flow problems and respective distributed solutions";Reza Asadi, Solmaz Kia; Journal of Automatica, Sinica</a></p>
